### ワンライナーWebサーバ

* [ワンライナーWebサーバを集めてみた](https://qiita.com/sudahiroshi/items/e74d61d939f18779970d)

    グローバル
    $ npm install -g http-server
    $ http-server -p 8000

    ローカル
    $ npm install http-server --save
    $ ./node_modules/.bin/http-server -p 8000
    $(npm bin)/http-server -p 8000
